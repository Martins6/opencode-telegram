# Overview

Telegram bot gateway that allows users to interact with an OpenCode AI agent directly from Telegram. The bot handles text messages, media uploads, slash commands, and restricts access to allowed users.

# Files

- feat-telegram-bot.md - Core Telegram bot with message handling, media processing, and user authentication
- feat-install-script.md - Installation script for easy binary deployment
- feat-opencode-run-integration.md - OpenCode run command integration replacing HTTP server
- feat-progress-updates.md - Progress updates for long-running operations
